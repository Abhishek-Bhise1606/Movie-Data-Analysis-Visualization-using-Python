# Movie-Data-Analysis-Visualization-using-Python


This project explores a movie dataset using *Python*, performing data cleaning, exploratory data analysis (EDA), visualization, and insights generation.
The notebook walks through univariate, bivariate, and time-series analysis to understand patterns in movie production, financial performance, ratings, and genre trends.

## 🚀 Features & Analysis Performed

### *1️⃣ Basic Exploration*

* Load and inspect dataset structure
* Check shape, columns, and data types
* Count unique movies, genres, directors, and lead actors
* Identify earliest and latest release years
* Summary statistics of numerical features
* Missing value detection

---

### *2️⃣ Univariate Analysis*

Includes several visualizations:

* Histogram of movie release years
* Top 5 most common genres
* Distribution of IMDb ratings
* Histogram of Rotten Tomatoes scores
* Average movie budget grouped by country

---

### *3️⃣ Bivariate Analysis*

Examines relationships between key variables:

* *Budget vs Global Box Office* correlation (Scatter Plot)
* *Average IMDb rating* across genres (Bar & Box plots)
* Impact of *budget on Rotten Tomatoes score*
* *US vs Global revenue* by genre
* Relationship between *Opening Day vs First Week sales*

---

### *4️⃣ Time-Based Analysis*

* Movie release trends over the years (Line Plot)
